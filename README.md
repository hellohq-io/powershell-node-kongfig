# [powershell-node-kongfig](https://hub.docker.com/r/hellohq/powershell-node-kongfig/)

**Docker image that includes `pwsh` and `kongfig`.**

## Installation

```sh
docker pull hellohq/powershell-node-kongfig
```

## Binaries

- `pwsh`: v6.0.2
- `node`: v10.6.0
- `npm`: v6.1.0
- `kongfig`: v1.0.3
